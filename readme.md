<h1>Projeto Literalura</h1>
<p>Este projeto Java + Spring consome a api gutendex para montar
uma série de funcionalidades que vai
permitir salvar os livros em base de dados própria
com PostgreSQL, a fim de armazenar e gerar dados
a partir destes livros.</p>
<h2>Funcionalidades:</h2>
<ul>
<li>Busca de livro por título</li>
<li>Exibir livros registrados</li>
<li>Exibir autores registrados</li>
<li>Exibir autores vivos em ano específico</li>
<li>Exibir livros de idioma específico</li>
<li>Top 10 mais baixados</li>
<li>Soma de downloads de todos os livros registrados</li>
<li>Buscar autor por nome na base de dados</li>
</ul>
